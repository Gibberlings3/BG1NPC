Généralités

Ce mod développe la profondeur du registre de personnalité et le niveau d'interaction des PNJs du jeu BG. Quand BG2 est sorti, une des principales améliorations était, aux yeux de beaucoup de personnes, le niveau accru d'interaction que l'on pouvait avoir avec les membres du groupe. Ce mod a été développé pour apporter aux joueurs de BG une expérience similaire. Les PNJs de votre groupe bavardent désormais avec le PJ, entre eux, et ont de petites quêtes secondaires personnelles. Comme nous n'avons pas été en mesure de faire enregistrer les nouvelles répliques par les acteurs de doublage d'origine, nous avons cherché à la place des musiques adaptées à chaque PNJ, destinées à être jouées à chaque fois qu'ils initient un bavardage. Cet ensemble musical est un paquetage séparé et optionnel (afin de garder faible la taille du téléchargement) intitulé The BG1NPC Music Pack. Vous pouvez le trouver sur la page de téléchargement de Gibberlings Three, ici.

Compatibilité

Ce mod est conçu pour fonctionner avec Tutu v4, EasyTutu et BGT, tous basés sur des installations complètes et patchées de Baldur's Gate et Baldur's Gate II. Ni l'extension Le Trône de Bhaal de BG2 ni l'extension La Légende de l'Île perdue de BG ne sont requises [NdT : pour BG1 NPC lui-même]. Depuis la v21 il est aussi compatible avec BGEE.

Ce mod ne s'installera pas correctemment avec des versions de WeiDU antérieures à v237. Cette distribution est livrée avec WeiDU v237, qui mettra automatiquement à jour votre version de WeiDU quand vous installerez le Projet BG1 NPC.

BG1 NPC n'est pour l'instant pas compatible avec Tutu v5 ou Tutu v6. Il se pourrait qu'il s'installe, il est certifié qu'il fonctionnera mal. Ne venez pas vous plaindre que nous n'avons pas prévenu. Si vous avez des versions historiques de ces rééditions, ne vous en servez pas ; EasyTutu et Tutu v4 sont les dernières plates-formes confirmées comme étant opérationnelles.

Sword Coast Stratagems, EasyTutu, BGT, BG1 NPC et BG Unfinished Business ont fait des efforts particuliers pour s'assurer que les joueurs peuvent utiliser ces mods ensemble sans conflit. En outre, nous avons adapté notre code pour permettre l'usage simultané de Sirine's Call [NdT : l'Appel des sirènes]. Veuillez visiter les forums pour plus d'informations, en particulier sur l'ordre d'installation.

Une discussion sur l'ordre d'installation et une liste de mods compatibles se trouvent dans The Tutu State of the Modding Union [NdT : "L'état de l'Union", de Tutu] chez les Gibberlings Three.

Pour BGT, on peut trouver une discussion sur l'ordre d'installation et une liste de mods compatibles en consultant les forums Spellhold Studios : compatibilité et en cherchant des recommandations particulières d'ordre d'installation ailleurs dans les forums, notamment dans Mega Mod Help et les projets comme BiG World Project (BWP). Même si vous ne jouez pas avec une installation Mega Mod, ce sont les endroits où les gens ont fait des expériences sur l'ordre d'installation de BG1NPC sur un jeu avec BGT.

Pour BGEE, vous pouvez trouver une liste de mods compatibles en consultant la liste de mods BGEE tenue par mlnevese sur les forums Baldur's Gate et chercher des recommandations sur l'ordre d'installation particulier ailleurs sur les forums.

Installation


Windows
Le Projet BG1 NPC est livré et installé avec WeiDU et est diffusé sous forme d'archive auto-extractible. Pour l'installer, il suffit de double-cliquer sur l'archive et de suivre les instructions à l'écran. [NdT : cliquez sur le bouton Accept, puis sur le bouton Browse pour sélectionner le répertoire du jeu et enfin cliquez sur Install.]

Autrement vous pouvez extraire les fichiers de l'archive à l'aide de WinRAR. Après une extraction réussie, vous devriez avoir un répertoire "bg1npc" et un fichier setup-bg1npc.exe dans le répertoire de votre jeu (le dossier qui contient le fichier CHITIN.KEY). Pour installer, il suffit de double-cliquer sur setup-bg1npc.exe et de suivre les instructions à l'écran.

Veuillez exécuter setup-bg1npc.exe dans le répertoire de votre jeu afin de réinstaller, désinstaller ou sinon pour changer des composants.

Mac OS X
Si le mod était déjà installé, désinstallez-le avant d'extraire la nouvelle version. Le Projet BG1 NPC est livré et installé avec WeiDU. Pour l'installer, il suffit d'extraire le contenu du mod dans le dossier de votre jeu puis de copier le contenu du dossier "OSX-bg1npc-v21" dans le dossier de votre jeu (le dossier qui contient le fichier CHITIN.KEY). Suite à une extraction réussie, vous devriez avoir un dossier "bg1npc" et des fichiers setup-bg1npc et setup-bg1npc.command dans le dossier de votre jeu. Pour installer, il suffit de double-cliquer sur setup-bg1npc.command et de suivre les instructions à l'écran.

Linux
Extrayez le contenu du mod dans le dossier du jeu que vous souhaitez modifier. Téléchargez la dernière version de WeiDU pour Linux à partir de WeiDU.org et copiez WeiDU et WeInstall dans /usr/bin. Ensuite ouvrez un terminal et allez dans le dossier d'installation de BG2, lancez 'tolower' et répondez Y aux deux questions. Vous pouvez éviter d'exécuter la seconde option (Linux.ini) si vous l'avez déjà lancée une fois sur le même dossier. Pour gagner du temps, l'archive est déjà passée en minuscules, aussi il n'est pas non plus nécessaire d'exécuter la première option (passer les noms de fichiers en minuscule) si vous n'avez extrait que ce mod depuis la dernière fois que vous avez passé les noms de fichiers en minuscule. Si vous avez un doute, exécuter tolower et choisir les deux options est sans danger.
Lancez WeInstall bg1npc depuis le dossier de votre jeu pour installer le mod. Puis lancez wine BGMain.exe et commencez à jouer.

Composants

Le Projet BG1 NPC contient 28 composants distincts qui peuvent être installés indépendamment les uns des autres.

Au début de l'installation, WeiDU demandera si vous voulez afficher quatre groupes de composants. Si vous souhaitez installer ou désinstaller n'importe lequel de ces composants, répondez "Y" ; sinon, l'installateur vous permettra de sauter directement à la section suivante.

Pour désinstaller complètement BG1 NPC, désinstallez le composant BG1 NPC Project : modifications nécessaires. Le reste du mod a besoin de la présence de ce composant de base, par conséquent il entraînera la désinstallation automatique de la totalité de BG1 NPC. Afin de vous assurer que les derniers éléments sont effacés, supprimez le répertoire bg1npc et les fichiers bg1npc.tp2 et setup-bg1npc.exe du répertoire d'installation.

Si vous souhaitez utiliser seulement les portraits ou les altérations, vous devez quand même installer ce composant de base de réparation.

Kagain, Coran, Garrick, Kivan, Dynahéir, Xzar, Jaheira, Tiax, et Xan ont des quêtes personnelles sur mesure. De petites rencontres "surprise" supplémentaires sont aussi incluses.

Ce composant installe soit le journal de Sarevok enrichi par SixofSpades, soit la version originale du Projet BG1 NPC.

original du jeu ;

BG1 NPC Project : vérification de la présence de Walking Speeds de TutuFix
Tutu uniquement

Si vous avez ajouté d'autres mods, une meilleure méthode consiste à installer le composant Walking Speeds de TutuFix après que tous les mods qui ajoutent de nouvelles quêtes, de nouveaux personnages ou PNJs aient été installés.

ATTENTION : le composant Walking Speeds de TutuFix a des effets secondaires sur les sorts Métamorphose, Hâte et Lenteur. Des corrections de ces comportements sont en cours de développement par l'équipe de BG2 Tweaks sur www.gibberlings3.net.

BG1 NPC Project : dialogues initiés par le joueur
Ce composant donne au joueur l'option d'engager le dialogue avec tous les membres du groupe. Ceci permet au PJ de poser des questions à propos d'un PNJ et de son passé. Il comprend aussi un correcteur de textes à l'initiave du joueur pour tous les PNJs de BG. Si un PNJ commence à prononcer des répliques d'un autre personnage ou a un problème de doublage, alors parlez au personnage et faites lui savoir que sa voix "semble bizarre". Le correcteur de textes des DIJ va réaffecter les doublages et réponses sonores du personnage, vous permettant de poursuivre la partie. Pour activer les DIJs ou le correcteur de textes, faites parler votre personnage au PNJ en question. Remarque : les DIJs doivent être installés après tous les autres composants du BG1 NPC. Si vous modifiez l'installation d'un composant impliquant des dialogues, ce composant sera automatiquement désinstallé et il se peut que vous ayez à le réinstaller manuellement.

Notes pour les modeurs

La philosophie guidant le codage dans ce projet est de préserver la compatibilité avec autant de plates-formes et de mods que possible. Le Projet BG1 NPC incorpore les corrections de base nécessaires au support du code du Projet, telles qu'indiquées ci-après :

  • Index des variables utilisées dans le code du Projet BG1 NPC, y compris le nom de fichier et le numéro de ligne
  • Utilisation de variables de création de mod multi plates-formes pour obtenir un code d'installation unique pour plusieurs plates-formes
  • Utilisation de variables "Fluid State" pour obtenir un code d'installation unique pour un codage compatible BGT des PNJs recrutables qui empilent les dialogues de BG après les dialogues de BG2. Cela s'avèrera particulièrement utile pour une utilisation en conjonction avec les versions 1.06 et supérieures de BGT
  • Patch .tp2 non destructif de pdialog.2da pour incorporer les scripts de rêve (Dream Scripts) pour tous les Personnages Non-Joueur standard de BioWare
  • Patch .tp2 non destructif de interdia.2da pour incorporer le fichier de bavardage (banter) d'Imoen s'il n'est pas présent
  • Initialisation non destructive (s'installe si pas présent) de bimoen.dlg/_bimoen.dlg
  • Patch .tp2 non destructif de happy.2da pour permettre la restauration des interjections Heureux des Neutres par CamDawg
  • Tutu uniquement : patch .tp2 non destructif de songlist.2da pour permettre l'échange de musique
  • Effacement de la répétition des dialogues par défaut de BioWare par ADD_STATE_TRIGGER et ADD_TRANS_ACTION à la plupart des fichiers (c.f. fichiers BG1NPC/Core/DLG/X#BGREPLACE_TUTU.D et BG1NPC/Core/DLG/X#BGREPLACE_BGT.D)
  • Support de l'installation sur Tutu v4 et SoA seul de cutspy.cre par la création d'un x#cutspy.cre qui fonctionne
